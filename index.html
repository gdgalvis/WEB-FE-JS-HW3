1. JavaScript Async

JavaScript Async refers to the ability to perform non-blocking operations. JavaScript is single-threaded, meaning it can only do one task at a time. However, using asynchronous code, JavaScript can handle multiple tasks simultaneously without waiting for each one to complete before moving to the next.

Example:


                console.log('First Test');
                setTimeout(() => console.log('Second Test'), 2500);
                console.log('Third Test');
                // Output: 
                // First
                // Third
                // Second (after 1 second)

2. Callback

A callback is a function passed as an argument to another function. It allows a function to execute after another function has finished executing, thus ensuring the proper order of operations. In other words it's a function which is to be executed after another function has finished execution.

Example:


                    function greet(name, callback) {
                        console.log('Hello Profesor ' + name);
                        callback();
                    }
                    
                    function sayGoodbye() {
                        console.log('Goodbye');
                    }
                    
                    greet('Octavio', sayGoodbye);
                    
                    // Output:
                    // Hello Profesor Octavio
                    // Goodbye

3. Promise

A Promise is an object representing the eventual completion or failure of an asynchronous operation. It provides a method to handle asynchronous code in a more readable and maintainable way.

Example:


                    let promise = new Promise((resolve, reject) => {
                        let success = true;
                        if (success) {
                            resolve('Test Successful');
                        } else {
                            reject('Test Failed');
                        }
                    });
                    
                    promise.then(result => {
                        console.log(result);
                    }).catch(error => {
                        console.log(error);
                    });
                    
                    // Output: 
                    // Test Successful

4. Async/Await

Async/Await is a syntactical sugar built on top of Promises, providing a more straightforward way to work with asynchronous code. The async keyword makes a function return a Promise, while await pauses the function execution until the Promise is resolved.

Example:


                    async function fetchData() {
                        try {
                            let response = await fetch('https://jsonplaceholder.typicode.com/todos/1');
                            let data = await response.json();
                            console.log(data);
                        } catch (error) {
                            console.log('Error:', error);
                        }
                    }
                    
                    fetchData();
                    
                    // Output:
                    //{
                    //  "userId": 1,
                    //  "id": 1,
                    //  "title": "delectus aut autem",
                    //  "completed": false
                    //}
                                    

fetchData();

5. Event Loop

The Event Loop is a mechanism that allows JavaScript to perform non-blocking operations by offloading operations to the system kernel whenever possible. The Event Loop continuously checks the call stack and the callback queue, executing tasks from the queue after the call stack is empty.

Example:


console.log('Start');

setTimeout(() => {
    console.log('This runs last');
}, 0);

console.log('End');

// Output:
// Start
// End
// This runs last
